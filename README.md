# 4d-plugin-apple-window-title-bar
Make window content view display behind title bar

To use on v16 or v17, move manifest.json to contents.

## Syntax

```
HIDE WINDOW TITLE BAR (window)
```

Parameter|Type|Description
------------|------------|----
window|LONGINT|
